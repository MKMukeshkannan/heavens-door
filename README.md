## How to Build
### Prerequisites
* C++23 or later
* CMake v3.30 or later
* Raylib installed on your system

### Build (example using CMake)
```bash
cmake -S . -B build
cmake --build build
```

### Run
```bash
./build/bin/heavens_door
```
